# MandelbrotSet
A simple Mandelbrot fractal renderer written in C++ using SDL and OpenGL 3.3
Includes a GPU renderer and a slow, single-threaded CPU renderer.

Controls:
WASD - Moving
Q - Decrease iteration number
E - Increase iterarion number
"Numpad +" - Zoom in
"Numpad -" - Zoom out
Esc - Quit
